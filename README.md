# Tsugai

Tsugai（番）は、キーボード操作を中心にした2画面ファイラーです。
左右2つのペインを並べ、片方からもう片方へコピー・移動しながらファイルを整理できます。

## ダウンロード

最新版は GitHub の [Releases](https://github.com/drunk-monkey-suiken/tsugai/releases) からダウンロードしてください。

環境に合わせて以下を選びます。

- Windows: `Tsugai-vX.Y.Z-win-x64.zip`
- macOS: `Tsugai-vX.Y.Z-macOS-universal.zip`

ZIP を展開して実行してください。

## 動作環境

### Windows 版

- Windows 11 x64
- Windows 10 はサポート中の Enterprise・LTSC・IoT 系を対象
- 配布形式: 自己完結 exe
- .NET ランタイムの別途インストールは不要です

初回起動時に Windows SmartScreen の確認が出る場合があります。現在の配布物はコード署名していないためです。信頼できる配布元から取得したことを確認してから実行してください。

### macOS 版

- macOS 14 Sonoma 以降
- Universal アプリ（Apple Silicon / Intel）

配布物は Developer ID 署名 / notarization をまだ行っていません。初回起動時に Gatekeeper の確認が出る場合があります。

#### Gatekeeper で開けない場合

通常は Finder で `Tsugai.app` を右クリック（または Control + クリック）して `開く` を選び、確認ダイアログでもう一度 `開く` を選ぶと起動できます。

それでも「壊れているため開けません」などが出る場合は、信頼できる配布元から取得したことを確認したうえで quarantine 属性を外してください。

```sh
xattr -dr com.apple.quarantine /Applications/Tsugai.app
open /Applications/Tsugai.app
```

zip の段階で外す場合は、展開前に次のようにします。

```sh
xattr -d com.apple.quarantine Tsugai-vX.Y.Z-macOS-universal.zip
unzip Tsugai-vX.Y.Z-macOS-universal.zip
```

## 主な機能

- **2画面コピー / 移動**: 反対ペインをコピー先・移動先として使います。
- **マーク操作**: `Space` で複数選択し、まとめてコピー・移動・削除・圧縮できます。
- **同名衝突処理**: コピー / 移動先に同名がある場合、上書き、末尾 `_` 付与、名前変更、キャンセルを選べます。
- **連番リネーム**: Windows / macOS ともに、複数マークして `R` を押すと、ベース名・開始番号・桁数・拡張子を指定して一括変更できます。
- **履歴ジャンプ**: Windows / macOS ともに、`H` で訪問済みフォルダの一覧を開き、過去の場所へ移動できます。
- **インクリメンタルサーチ**: `F` で一覧を絞り込まず、先頭一致する項目へカーソルを移動します。
- **中身検索（grep）**: `G` でフォルダ以下を再帰検索し、結果から該当ファイルや該当行へ移動できます。
- **画像プレビュー**: 画像ファイルは反対ペインにプレビュー表示できます。
- **書庫の閲覧 / 展開 / 圧縮**: 書庫を解凍せずに中へ入れます。書庫内は読み取り専用です。
- **パスワード付き zip**: 閲覧、展開、取り出しに対応します。パスワードは保存しません。
- **パスコピー**: ファイル名、フルパス、フォルダパスを選んでコピーできます。
- **ブックマーク / 場所ジャンプ / アプリ起動**: よく使う場所やアプリを一覧から選べます。
- **外部エディタ / ターミナル連携**: カーソル位置の項目をエディタで開いたり、現在フォルダでターミナルを開けます。
- **左右比較**: WinMerge / FileMerge などの比較ツールで左右ペインの対象を比較できます。
- **ネイティブコンテキストメニュー**: OS のメニューやシェル拡張を使えます。
- **設定保存**: キーバインド、ブックマーク、アプリ起動メニュー、外観、外部ツールなどを保存します。

## 中身検索（grep）

`G` でフォルダ以下を再帰検索できます。

- 結果から `Enter` で該当ファイルの場所へ移動
- `Shift` + `Enter` でエディタの該当行を開く
- `Space` で行表示 / ファイル表示を切替
- `Esc` で検索結果を閉じる

Windows 版、macOS 版ともに、設定で ripgrep などの外部検索ツールを指定できます。未設定時は内蔵スキャナを使います。

## 書庫

- macOS: zip / tar / tar.gz / tgz / tar.bz2 を扱えます。
- Windows: zip は内蔵で扱えます。7-Zip がある場合は 7z / rar / lzh / cab / iso / arj / tar.gz も扱えます。
- 書庫内は読み取り専用です。コピー操作は「取り出しコピー」として動きます。
- パスワード付き zip は閲覧・展開・取り出しに対応します。作成は対象外です。

## 設定ファイル

設定はユーザーフォルダ配下に保存されます。

```text
Windows: %APPDATA%\Tsugai\
macOS:   ~/Library/Application Support/Tsugai/
```

主な保存内容は、キーバインド、ブックマーク、アプリ起動メニュー、外観、外部エディタ / ターミナル、比較ツール、検索ツールです。
